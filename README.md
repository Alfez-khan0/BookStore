# 📚 BookStore Web App

## 🚀 Overview

A full-stack BookStore web application where users can browse books, add to cart, manage favourites, and place orders.

---

## 🛠 Tech Stack

* Frontend: React + Vite + Tailwind CSS
* Backend: Node.js + Express.js
* Database: MongoDB

---

## ✨ Features

* User Authentication (Login/Register)
* Browse all books
* Add to cart
* Add to favourites
* Place orders
* View order history
* Admin: Add / Update / Delete books
       : Change Order status

---

## ⚙️ How to Run

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
